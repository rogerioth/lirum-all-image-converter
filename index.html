About

Lirum All Image Converter

Version 1.0.0

A simple, lightweight desktop application for converting images between different formats. Supports modern formats like HEIC and AVIF through WebAssembly decoders.

lirumlabs.com

Supported Formats

Input: WebP, HEIC, AVIF, PNG, JPG, GIF, BMP, TIFF

Output: JPEG, PNG, WebP, AVIF, HEIC, GIF, BMP, TIFF

Menu Shortcuts

  • Ctrl/Cmd + O Open image file
  • Ctrl/Cmd + L View logs
  • Esc Reset / Close modal

Use the menu bar for more options: File, Edit, View, Window, Help
